\# ACB Large Print \& WCAG Warrior Auditor (CLI)



This tool is a strict protocol-based auditor designed to validate documents against the \*\*American Council of the Blind (ACB)\*\* Large Print standards and \*\*WCAG 2.1\*\* structural requirements. 



It ignores "AI-guessing" in favor of deterministic Python logic to ensure 100% compliance with industry-standard benchmarks (DAISY/W3C).



\## 🛡️ Compliance Protocols

1\. \*\*ACB-LP-01 (Font Size):\*\* Minimum 18pt/24px for all body text.

2\. \*\*ACB-LP-02 (Leading):\*\* Minimum 1.5 line-spacing.

3\. \*\*WCAG-1.3.1 (Hierarchy):\*\* No skipped heading levels (e.g., H1 to H3).

4\. \*\*WCAG-1.1.1 (Non-Text):\*\* Mandatory `alt` attribute presence for all images.

5\. \*\*MD-STRICT (Syntax):\*\* Strict Markdown validation (e.g., enforcing space after `#`).
